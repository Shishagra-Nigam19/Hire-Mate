# 🚀 HireMate: Your AI-Powered Career Copilot

**HireMate** is a state-of-the-art career guidance platform that leverages the power of Artificial Intelligence to help job seekers land their dream roles. From resume optimization to live web job searching, HireMate provides a suite of specialized AI agents designed to navigate the modern job market.

### 🔗 Live Links
*   **Frontend**: [HireMate on Vercel](https://hire-mate-26fv-3jjwxcofj-shishagra-nigam19s-projects.vercel.app/)
*   **Backend**: [API on Render](https://hire-mate-backend.onrender.com/)

---

## ✨ Key Features

| Feature | Description |
| :--- | :--- |
| **📄 Resume Optimization** | Tailor your resume to specific job descriptions using AI-driven insights. |
| **🎯 Interview Prep** | Conduct mock interviews with an AI coach that provides real-time feedback. |
| **🎓 Career Guidance** | Get personalized advice on career paths and industry transitions. |
| **🔍 Live Web Job Search**| Scour the real-time web for job postings that match your specific skill set. |
| **📊 ATS Score Checker** | Simulate an Applicant Tracking System to see how well you rank for a role. |
| **🧩 Skill Gap Analysis** | Identify exactly what you need to learn to qualify for your target position. |

---

## 🛠️ Tech Stack

### Frontend
*   **Next.js 15 (App Router)**: High-performance React framework.
*   **React 19**: Utilizing the latest experimental features and hooks.
*   **Clerk Authentication**: Secure and seamless user management.
*   **Glassmorphism UI**: Modern, premium aesthetics with vibrant gradients.
*   **Lucide Icons**: Crisp, professional iconography.

### Backend
*   **FastAPI**: Blazing fast Python web framework.
*   **Google Gemini AI**: Powered by `gemini-2.0-flash` for intelligent assistance.
*   **SQLAlchemy**: Robust ORM for user data management.
*   **Pypdf**: Advanced PDF parsing for resume analysis.

## ⚙️ Installation & Local Setup

### Backend
1. `cd backend`
2. `python -m venv venv`
3. `venv\Scripts\activate`
4. `pip install -r requirements.txt`
5. `python -m uvicorn main:app --reload`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
